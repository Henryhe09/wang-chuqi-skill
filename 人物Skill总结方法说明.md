# 王楚淇 Skill 仓库结构与人物 Skill 总结方法说明

本文档说明本目录如何按照 `feng-ge-skill` 的方法组织一个人物视角 skill。

## 2. 总结方法

本 Skill 不把直播切片语录直接搬进 `SKILL.md`，而是按 7 个维度拆解公开素材：

| 文件 | 研究维度 | 作用 |
| --- | --- | --- |
| `01-writings.md` | 核心论点与语录 | 提取反复出现的足球观点、短句、赛后情书和核心判断。 |
| `02-conversations.md` | 长对话与即兴思维 | 分析直播回复、转写内容和被追问时的思路。 |
| `03-expression-dna.md` | 表达风格 DNA | 总结称谓、开头、反问、节奏、赛后情书、幽默和禁忌。 |
| `04-external-views.md` | 外部视角与争议 | 记录平台资料、节目报道、百科履历和第三方转载。 |
| `05-decisions.md` | 决策记录与行动模式 | 从退役转型、直播表达、节目参与中提取行动规则。 |
| `06-timeline.md` | 时间线 | 建立履历、内容演进和调研时间边界。 |
| `07-terminology.md` | 黑话术语体系 | 记录"淇"、"淇圣"、"直播回复"、"赛后情书"、"出列"等社区术语和用法。 |

## 3. 从素材到 Skill 的压缩规则

- 先把素材分为公开事实、直播观点、第三方评价、用户补充素材和推断。
- 再从重复模式中提炼心智模型，而不是只模仿口头禅。
- 每个心智模型都写清楚"一句话、证据、应用、局限"。
- 表达 DNA 作为核心外壳，尤其保留直播切片、嘴硬否认和赛后情书风格。
- 对实时赛况、转会、伤病、私人生活和博彩建议写清楚边界。

## 4. 本 Skill 的核心提炼

`SKILL.md` 最终压缩为 7 个心智模型：

- 亲历优先：先问球员怎么活。
- 联赛环境 > 青训口号。
- 分场景说话：直播狠，解说收。
- 稳步接住流量。
- 反营销号：别只看标签，看过程。
- 专业娱乐化：梗是入口，足球是底盘。
- 赛后情书：把战报写成史诗爽文。

## 5. 使用边界

这个 Skill 能提供"王楚淇式足球直播/赛后情书视角"，但不能代表本人当前观点。遇到 2026 年 4 月之后的赛事、转会、账号数据或监管事件，应重新检索最新公开资料。
